GitHub prisijungimas — reikalingas tik norint išsaugoti pakeitimus visiems
Kaip susikurti žetoną:
1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token.
2. Repository access: pasirink tik šitą repo (Kartelis-Torpedos).
3. Permissions → Contents → Read and write.
4. Sukurk, nukopijuok ir įklijuok čia.

Žetonas laikomas tik šios naršyklės kortelės atmintyje (sessionStorage) — jis nepatenka į GitHub failus ir išsivalo užvėrus skirtuką. Niekam jo nesiųsk ir neįkelk į repo. Jei pamesi prieigą — gali bet kada sukurti naują ir seną panaikinti (Revoke).
Pati svetainė (HTML/CSS/JS) ir data.json failas turi būti tame pačiame GitHub repo, įjungtas GitHub Pages. Kai paspaudi „Išsaugoti į GitHub" narystės lange, pakeitimai įrašomi tiesiai į data.json — visi, kas atsidaro svetainę, pamatys atnaujintus duomenis po kelių minučių (kol atsinaujina GitHub Pages).
